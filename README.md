# MeNabung Contracts

DeFi vault that splits IDRX across yield strategies on Base.

## Quick Start

```bash
forge install
forge build
forge test
```

## Contracts

- `MeNabungVault.sol` - Main vault
- `ThetanutsAdapter.sol` - Options strategy
- `AerodromeAdapter.sol` - LP strategy
- `StakingAdapter.sol` - Staking strategy
